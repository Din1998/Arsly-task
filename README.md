# Front-End Developer GSAP Animation Task

##  Overview
This project is a submission for the **Front-End Developer Assignment**.  
The goal is to implement **modern, smooth, and premium GSAP animations** based on provided UI designs.

The focus is not just animation, but also:
-  Smoothness & timing
-  Premium UI/UX feel
-  Performance optimization
-  Clean and structured code
